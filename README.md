## 🖼️ Ajouter des images

Dépose tes images dans le dossier `images/` (formats recommandés : `.jpg`, `.png`, `.webp`).

Dans ton HTML, appelle-les ainsi :
```html
<img src="./images/nom-de-ton-image.jpg" alt="Description" />
```
