# Dossier Images

Dépose tes images dans ce dossier.

**Formats recommandés :** `.jpg`, `.png`, `.webp`

**Utilisation dans l'HTML :**
```html
<img src="./images/nom-image.jpg" alt="Description de l'image" />
```

**Exemples de noms :**
- `photo-profil.jpg`
- `anteus-instagram-1.jpg`
- `projet-1.png`
